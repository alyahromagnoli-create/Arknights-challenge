# 终端代号：Arknights Protocol (明日方舟随机战术挑战生成器)

欢迎登入 PRTS 系统。本项目是一个带有浓厚《明日方舟》UI 风格的随机战术挑战生成网页，专为寻求极限作战体验的博士打造。

## 🌟 核心功能 (V4.0 2026 Edition)

* **双轨战术协议**：支持 `EASY (基础)` 与 `HARD (极限)` 难度无缝切换，分别对应不同的变态限制词条。
* **海量目标数据库**：收录至最新主线高难关卡、往期 SS 经典关卡 (EX/S关) 以及四大集成战略（水月/银凇/无终/界园）的多层级难度 (N8~N15)。
* **战术封锁机制 (Blacklist 10)**：与时俱进的 Ban 位系统，涵盖至 2026 年最新核心/幻神干员，每次生成随机封锁 10 名干员。
* **沉浸式交互视觉**：深度还原罗德岛终端的切角 UI、科幻扫描线滤镜以及数据破解（乱码解析）动画。

## 🛠️ 本地运行与自定义扩充

**本地运行**：
本项目采用纯静态前端技术栈，无需配置服务器。直接下载本仓库的 `index.html` 文件，用任意现代浏览器（Chrome/Edge/Safari 等）双击打开即可使用。

**如何自定义词库**：
如果你想添加自己喜欢的干员或更离谱的规则，可以直接用记事本或代码编辑器打开 `index.html`，找到代码中的 `databases` 对象进行修改：
* `maps`: 扩充作战目标关卡。
* `easyRules` / `hardRules`: 自定义基础与极限限制条件。
* `banned`: 添加你需要针对的强力干员名单。

*修改并推送到 GitHub 后，Vercel 会自动为你部署更新。*
